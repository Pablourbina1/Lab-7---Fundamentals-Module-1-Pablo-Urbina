# Region Filter 
## Parte 1 del laboratorio
#### Link del video de explicación: 
En esta parte se añadio una opción para filtrar los paises por region, haciendolo de manera eficaz realizando una sola llamada a la API y generando todas las regiones con dicha llamada
para asegurar una ejecución fluida del programa, a la vez de que se añadio la opción para ver los paises de todas las regiones sin filtros, funcionando de la mano con el search input.

## Definition of Done
| Criterio               | Descripción                                                         |
|------------------------|---------------------------------------------------------------------|
| Existe dropdown        | Elemento `<select>` con opciones de región                          |
| Regiones cargadas      | Opciones obtenidas de la API (África, Américas, Asia, Europa, Oceanía) |
| Filtro funciona        | Al seleccionar una región se muestran solo los países correspondientes |
| Filtros combinados     | La búsqueda y el filtro por región funcionan juntos                 |
| Opción de reinicio     | "Todas las regiones" muestra todos los países                       |
| Estado preservado      | El filtro se mantiene al escribir en la búsqueda                    |
| Sin errores            | Consola libre de errores                                            |
